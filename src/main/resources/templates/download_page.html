Reaper script Note names

Internet connection required

    The script for Reaper is quite basic. All processing and data are handled on the server. The script communicates with the main application over the internet, so a reliable internet connection is required.

Installation

  • In Reaper, click to Options -> Show Reaper resource path in explorer/finder.
  • The "satbify.lua" file should be placed inside the "Scripts" folder.
  • The "satbify_note_names.txt" file should be placed inside the "MIDINoteNames" folder.
  • When moved, in Reaper's Actions list, add the script, and assign any shortcut.
  • Open the MIDI editor, go to File -> Note/CC name -> Load note/CC name from file -> Choose file... -> select "satbify_note_names.txt" file.
